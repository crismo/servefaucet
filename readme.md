## Install

Ensure you have [Node.js](https://nodejs.org) version 8 or higher installed. Then run the following:

```
$ npm i -g https://github.com/crismo/serverfaucet.git
```

## Usage

```
$ serverfaucet 

  Usage
    $ serverfaucet [port] (default 8080)

  Example
    $ serverfaucet 81
    Faucet Listener
	    http://127.0.0.1:81
	    http://10.20.48.10:81

```
